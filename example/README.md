# 面试题功能实现说明 (README)

## 修改文件
`example/demo.html`

## 实现功能清单
1. **鼠标交互**
   - **显示**: 鼠标移入节点 (`jmnode`) 时，自动显示数据编辑窗口。
   - **隐藏**: 点击画布空白处或页面其他非编辑区时，窗口自动隐藏。
   - **定位**: 窗口位置根据节点位置自动计算，并包含简单的边缘检测防止溢出屏幕。

2. **键值对管理**
   - **展示**: 窗口根据当前节点的 `data.customFields` 数据动态渲染键值对列表。
   - **添加**: 在底部输入框填写 Key 和 Value，点击 "Add Key-Value" 按钮添加。
   - **删除**: 点击条目右侧红色的 "X" 按钮即可删除对应键值对。

3. **数据编辑与持久化**
   - **修改**: 直接在列表的输入框中修改 Value 值。
   - **自动保存**: 监听 `input` 事件，修改内容会实时同步到内存中的节点数据对象，无需手动点击保存。

4. **数据隔离**
   - 所有自定义数据存储在节点的 `data.customFields` 属性中。
   - 每个节点的数据完全独立，互不干扰。

## 启动与运行步骤

建议使用本地服务器运行以避免浏览器跨域限制（CORS）。

1. **进入项目目录**
   打开终端（Terminal），确保位于项目根目录：
   ```bash
   # 进入项目文件夹 (假设文件夹名为 jsmind)
   cd jsmind
   ```

2. **安装依赖**
   首次运行需安装依赖：
   ```bash
   npm install
   ```

3. **启动服务器**
   运行内置的 http-server：
   ```bash
   npm run server
   ```

4. **预览页面**
   打开浏览器访问以下地址：
   [http://127.0.0.1:8080/example/demo.html](http://127.0.0.1:8080/example/demo.html)

## 代码位置
核心逻辑位于 `example/demo.html` 文件底部的 `<script>` 标签内，主要函数为 `init_kv_editor`。
